One-time setup

There are three layers here:

  • Global — the CA pubkeys (below). Nothing to install; just referenced.
  • Per-fleet — the module file or sshd_config block. Drop it in once; it's the same content on every host.
  • Per-host — the configuration.nix import (with the host's signed cert) or the HostCertificate line. Different on each box.

The signing checklist at the bottom only matters if you've set ssh_ca.host_ca_key_file.
